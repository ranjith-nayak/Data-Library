## Repository Structure

Each major topic is organized into three folders based on **my current level of understanding**, not on any platform's difficulty rating.

```text
Topic
│
├── 01_Easy
├── 02_Medium
└── 03_Hard
```

As my knowledge improves, topics may move between these folders. This repository reflects my personal learning progress rather than fixed difficulty levels.

Each topic contains practice questions, explanations, examples, and notes collected throughout my learning journey.

## Learning Philosophy

I believe real learning comes from understanding, not memorizing.

For every concept I try to:

* Understand the idea.
* Solve problems.
* Explain the solution in my own words.
* Document key learnings.
* Revisit concepts regularly.

If I don't understand something, I'll keep practicing until I do.
